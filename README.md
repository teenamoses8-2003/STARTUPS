-- Startup Analytics Dashboard – Power BI Project

Recently, I worked on a Startup Analytics Dashboard using Power BI to improve my data analytics and dashboard designing skills. The idea behind this project was to analyze startup data in a simple and interactive way so users can understand funding trends, startup growth, valuations, employee strength, and business performance easily.

In this project, I used Power BI for creating dashboards, Power Query for cleaning and transforming data, and DAX for calculated measures and KPIs. I tried to design the dashboard in a clean and professional way with interactive visuals, navigation buttons, slicers, KPI cards, charts, and filters.

Some of the key insights covered in this dashboard are:
• Total Funding Raised
• Startup Valuation Analysis
• Monthly Active Users (MAU)
• Employee Growth
• Industry-wise Startup Performance
• Funding Trends Over Time
• Top Startups by Valuation & Funding
• Growth Comparison Between Startups

While building this project, I learned a lot about:

* Data Cleaning
* Data Modeling
* Dashboard Designing
* DAX Measures
* Data Visualization
* Business Intelligence Concepts
* KPI Reporting

One of the most interesting parts of this project was creating comparisons between startups and finding patterns in funding and growth trends. This helped me understand how dashboards can convert raw data into meaningful business insights.

This project also helped me improve my storytelling skills with data and gave me hands-on experience with real-world analytics dashboards.

Tools Used:
* Power BI
* Power Query
* DAX
* Excel Dataset

I’m continuously learning and working on improving my skills in Data Analytics, Power BI, SQL, and Python by building practical projects like this.

Feedback and suggestions are always welcome.
